Acessar a API Sofia em https://sofia-api.meshwave.com.br/api/v1/oracle/manual para verificar se há novas missões. Se houver, executar a missão. Se não houver, aguardar e verificar novamente.
Análise inicial da API Sofia
Navegando para a URL da API Sofia para análise inicial.
Verifiquei a API Sofia e encontrei a Guia de Comunicação e Governança, que detalha a hierarquia de comando e procedimentos de consulta. Ainda não há informações específicas sobre o endpoint de missões. Próximo passo: explorar a API para identificar como listar e executar missões, seguindo as diretrizes de hierarquia.
Procurando a documentação oficial da API Sofia para identificar o endpoint correto para verificar missões.
Tentando acessar a base da API Sofia para encontrar outros endpoints.
Com base na busca, não encontrei documentação oficial que indique o endpoint para verificar ou listar missões na API Sofia. A navegação até a URL base também não revelou informações adicionais.
